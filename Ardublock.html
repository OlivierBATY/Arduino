ARDUBLOCK



Langage de programmation graphique.

Ardublock est un plugin qui s'ajoute à l’environnement de programmation Arduino.

Il permet de programmer des cartes Arduino graphiquement à partir de blocs, pour générer ensuite les lignes de codes en langage C.


Exécution d'Ardublock

Pour exécuter Ardublock il faut d'abord exécuter l'IDE d'Arduino.

Dans l'IDE d'Arduino, aller dans le menu Outils, puis cliquer sur Ardublock.
